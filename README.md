# Homework Assignments

This is a repository containing a homework assignment from the course Foundations of Computing at Columbia University Graduate School of Journalism.

## Assignment
* [Assignment 1] (Dataset ONE - Beer cans.ipynb) - An assignment to further practice panda basics by cleaning and analyzing a dataset about American beer.
* [Assignment 2] (Dataset TWO Dogs.ipnyb) An assignment to practice advanced panda skills by cleaning, mergings and analyzing multiple datasets about pets in NYC.
